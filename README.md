## Abstract
本研究では、企業等に自分を優秀な人材だと思わせるためのポートフォリオを作成する。具体的には、企業が私を雇いたいと思うようなWebページを作成する。それと並行して、このポートフォリオに書くための実績作りも行なっていく。
